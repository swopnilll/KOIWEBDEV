Objectives

The primary objective of Lab 1 is to introduce students to the fundamental concepts of HTML and CSS. By the end of the lab, students should be able to create a simple web page structure using HTML and apply basic styling with CSS.

Lab Activities

During the lab, students engaged in various activities, including creating an HTML document, adding headings, paragraphs, images, and links. They also learned to use CSS to style their web pages by modifying colors, fonts, and layout.

Key Concepts

Key concepts covered in this lab included the basic structure of an HTML document, common HTML elements such as headings, paragraphs, and images, as well as CSS selectors and properties to style HTML content effectively.
